# SUNSHINE WEB IITH

This is the official repository for the new Sunshine web platform at IITH.

### Tech Stack

* **Framework:** React (bootstrapped with Vite)
* **Language:** JavaScript
* **Package Manager:** npm
* **Node.js:** It is recommended to use the latest LTS (Long-Term Support) version of Node.js (e.g., Node.js 22.x or later).

### Installation

1.  **Clone the repository**

    ```bash
    git clone https://github.com/iith-sunshine-web-app/sunshine_web_iith.git
    cd sunshine-iith
    ```

2.  **Install dependencies**

    ```bash
    npm install
    ```

### Running the Development Server

To start the local development server, run:

```bash
npm run dev
